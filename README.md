# Amazon Sales Analysis Dashboard (Power BI)

## Project Overview

This project presents an end-to-end sales analysis of Amazon data using Power BI.
The goal is to identify key business insights, sales trends, and performance drivers to support decision-making.

---

## 🎯 Business Objectives

* Analyze overall sales performance
* Identify top-performing products and categories
* Track sales trends over time
* Improve decision-making using data-driven insights

---

## 🛠 Tools & Technologies

* Power BI (Data Modeling & Visualization)
* DAX (Data Analysis Expressions)
* Excel / CSV (Data Source)

---

## 📊 Dashboard Features

### 🔹 1. Sales Overview

* Total Sales
* Total Orders
* Profit / Revenue KPIs
* High-level performance summary

### 🔹 2. Product Analysis

* Top-selling products
* Category-wise performance
* Contribution to total revenue

### 🔹 3. Time-Based Trends

* Monthly / Yearly sales trends
* Seasonal patterns

---

## 📈 Key Insights

* Sales show significant growth during peak seasons
* A small number of products contribute to a large portion of revenue
* Trends indicate opportunities for targeted marketing
